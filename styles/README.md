# Fluxbox Custom Themes

These are the custom Fluxbox themes currently kept in:

`/home/wachin/.fluxbox/styles/`

The themes are grouped into five visual families:

- `Modern-Clear-Dusk`: dark graphite, teal focus, warm gold clock text, coral pressed-button accent.
- `Windows-7-Blue`: blue glass style inspired by the default Windows 7 look.
- `macOS-Aqua`: light macOS-inspired chrome with blue focus and soft graphite text.
- `Aurora-Slate`: blue graphite chrome, green aurora focus, amber clock text, coral pressed-button accent.
- `Windows-XP-Luna`: Windows XP Luna-inspired royal blue chrome, pale menus, amber task focus, and green grips.

## Notes

Created the compact variant here:

`/home/wachin/.fluxbox/styles/Modern-Clear-Dusk-Medium/theme.cfg`

Changes from the original:

- Window title height: `28` to `24`
- Toolbar height: `28` to `24`
- Menu row height: `26` to `23`
- Fonts: `Noto Sans-10` to `Noto Sans-9`
- Handle width: `4` to `3`

Created the small variant here:

`/home/wachin/.fluxbox/styles/Modern-Clear-Dusk-Small/theme.cfg`

Changes from the medium version:

- Window title height: `24` to `21`
- Toolbar height: `24` to `22`
- Menu row height: `23` to `21`
- Fonts: `Noto Sans-9` to `Noto Sans-8`
- Handle width: `3` to `2`

All theme families use the same size scale:

- Base theme: comfortable size.
- `Medium`: balanced compact size.
- `Small`: smaller text and controls so more applications fit in the taskbar.

## How To Apply

Select the theme from the Fluxbox styles menu.

If Fluxbox does not refresh immediately, choose `Reajustar Fluxbox` from the menu.
